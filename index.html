COINS

Cryptocurrencies are non-tangible digital currencies that are created through a process called mining. Mining is a form of computer processing that allows a single larger task to be performed in small chunks simulatneously amongst many "sub" machines. When the individual computer completes the process, that machine is awarded a fraction of a coin. In simple terms, this all equates to faster computing.

EXCHANGES

There are many exchanges that exist for trading. Exchanges are platforms which can be used to trade (buy or sell) cryptocurrency coins. Users buy coins as an investment with the hope that some day, the value of the coins they buy increase significantly. This is where selling usually occurs. Once a user has observed a growth in value they deem satisfacroty, he or she can use an exchange to sell for profit.
